# Changelog

All notable changes to this project are documented in this file.

## [0.3.1] - 2026-04-13
- Cleaned legacy repository residues to make structure easier to understand.
- Removed old OpenClaw skill/workspace directories from main branch.
- Removed duplicated historical iteration samples and kept focused acceptance sets.
- Updated quality eval target to a kept sample under iteration-5.

## [0.3.0] - 2026-04-13
- Added `novel scene simulate` (`novel ss`) as single-entry multi-cast simulation.
- Added `novel outline view` (`novel ov`) with `vertical|horizontal|both` modes.
- Added `outline-board` reference and updated quickstart/session kickoff docs.
- Expanded executable eval matrix to 6 cases including scene/outline features.
- Added iteration-5 workspace outputs for final acceptance.

## [0.2.0] - 2026-04-13
- Introduced `novel-coauthoring-standard` as agent-agnostic standard skill.
- Kept `novel-coauthoring-openclaw` as compatibility layer.
- Added agent integration guide, migration guide, and standard CI workflow.
- Added short command aliases for memorability.

## [0.1.0] - 2026-04-13
- Initial release of coauthoring workflow, contract docs, memory model, evals, CI.
