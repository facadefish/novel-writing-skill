# 大纲看板（横纵双维）

## 定位
- 作者已知目标与结局前提下的结构化可视化能力。
- 只做展现和风险标记，不改主线结论。

## 纵向视图（vertical）
- 按事件时序展示：触发 -> 冲突 -> 兑现 -> 后果。
- 每个事件节点显示：参与角色、世界规则约束、未回收伏笔。
- 连接关系显示：因果链、冲突升级链、收益兑现链。

## 横向视图（horizontal）
- 固定某一时间点查看系统态。
- 人物层：立场、关系张力、资源状态。
- 故事层：主线推进度、支线占比、风险点。
- 世界层：规则触发状态、秩序压力、外部威胁。

## 双维联动（both）
- 左侧纵向线，右侧横向快照。
- 选中任一事件后，横向面板同步该时点状态。
- 自动标记断链事件、OOC 高风险角色、伏笔遗失项。

## 推荐输出结构
- `outline.vertical[]`
- `outline.horizontal`
- `outline.links[]`
- `risk_flags[]`
