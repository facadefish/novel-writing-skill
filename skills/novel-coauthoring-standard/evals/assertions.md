# 评测断言建议

执行基准文件：`evals/executable-matrix.yaml`

## Eval 1：角色重做
- 断言 1：输出包含至少 3 套方案
- 断言 2：每套方案包含“主线影响”说明
- 断言 3：输出包含风险与下一步动作

## Eval 2：节奏诊断
- 断言 1：输出包含段落级问题定位
- 断言 2：输出包含可替换节奏骨架
- 断言 3：正文示例为自然段落而非条目

## Eval 3：Agent 重放
- 断言 1：输出含 request_id 与 session_id
- 断言 2：输出含 idempotent_replay 字段
- 断言 3：输出能给出失败回退建议

## Eval 4：群像场景模拟
- 断言 1：输出包含回合级对白、动作、行为信息
- 断言 2：输出包含 OOC 风险点与替代建议
- 断言 3：输出不改写作者给定的场景目标与结局

## Eval 5：大纲看板
- 断言 1：输出同时包含纵向事件线与横向时点系统态
- 断言 2：输出包含事件关联或关系链接信息
- 断言 3：输出包含断链或高风险标记
