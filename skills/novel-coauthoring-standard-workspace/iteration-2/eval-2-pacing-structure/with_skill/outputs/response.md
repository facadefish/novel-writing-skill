# 结论摘要
你要的不是“段落润色”，而是“爽点编排重构”。这章应从“解释型后半章”改成“压迫-代价-兑现-抬钩”的四拍结构，并在正文中形成连续情绪曲线。

## 全章结构重排（后半章）
- A 段（压迫升级，约 20%）：敌方完成合围，主角资源见底。  
- B 段（代价抉择，约 25%）：主角必须在“保命/保同伴/保秘密”三选一。  
- C 段（爽点兑现，约 35%）：主角用有代价的非常规手段反击，出现“看似险胜”的快感。  
- D 段（后遗症+新钩，约 20%）：反击成功但付出后遗症，门后秘密抬升到下一章。

## 爽点节奏设计
- 小爽点 1：主角识破对手最强一招的破绽。  
- 大爽点：主角牺牲短期战力换取战局翻盘。  
- 余震钩子：翻盘并非终局，真正危险来自“门后机制”。

## 正文示例 1（A->B 段）
血池边缘的石纹一寸寸亮起，林凡才发现自己退无可退。三名执刀者封死了外圈，剩下那名灰袍老者不急着出手，只把指尖轻轻压在阵眼上，像是等他自己把最后一口气耗尽。林凡低头看了一眼掌心，护脉符只剩最后一枚，若用在自己身上，他能活；若用在身后那名重伤弟子身上，他就再没有第二次出刀的机会。

他听见身后咳血声断断续续，像有人在黑夜里敲一面破鼓。灰袍老者笑了笑，说你不是一向算得很清楚吗，今日也算一算，谁该先死。林凡没有立刻答话，只把护脉符攥得更紧，直到指节泛白。他知道这一息必须选，选错了不只是输这一战，而是以后每次回头都会看见同一双眼睛。

## 正文示例 2（C 段）
下一瞬间，林凡把护脉符按进了重伤弟子的胸口，自己反手拔刀，直接踏进阵眼。灵力逆流像铁钩一样把经脉一寸寸扯开，他几乎听见骨头深处的裂响，可刀势却在那一刻快到极致。灰袍老者本以为他会保命，出手慢了半拍，这半拍就够了——刀锋斜切而上，先断阵纹，再破护罡，最后在老者肩头留下了一道从锁骨拖到后背的血线。

围杀的人同时后撤，谁也没想到他会拿自己的后路换这一刀。林凡落地时膝盖一软，嘴里全是铁锈味，却仍把刀尖抵在地上没有倒下。他抬眼看向对面，第一次看见灰袍老者神色里有了惧意。那不是赢了之后的轻松，而是“你们终于知道我会拿什么去换”的冷硬宣告。

## 正文示例 3（D 段）
阵势散开后，血池底部传来一声极轻的闷响，像有第二道门在水下缓慢开启。林凡刚想迈步，右臂却猛地失去知觉，整条手臂像被冻在冰里，再也抬不起来。他这才明白，刚才那一刀不是代价“会不会来”，而是代价“什么时候爆发”。

重伤弟子被扶起时，低声说门后有东西在叫你的名字。林凡没有回话，只盯着池底那道逐渐扩大的黑缝。他赢下了这一场，但下一场已经提前到了门后，而且这一次，他未必还有一只完整的手可以出刀。

## 建议下一步
1. 把原后半段两段设定解释并入 A 段动作里。  
2. 在 B 段明确“三选一代价”，让读者看到不可逆选择。  
3. C 段后立刻给身体后遗症，不要拖到下章开头。  
4. D 段钩子同时绑定“世界观秘密 + 主角战力受损”，提升追更黏性。
