# 结论摘要
女主“工具人化”的根因是目标依附主角、决策缺乏代价、成长节点不独立。建议改为“自带主线目标 + 与主角阶段同盟/冲突并存”的双线角色。

## 可选方案
### 方案 A：复仇驱动型
女主目标是查清宗门灭门真相，短期与主角联手，中期因线索分歧产生冲突，后期在共同敌人前再度结盟。  
主线影响：能强化“信息差”与悬疑张力，适配玄幻中后期势力博弈。

### 方案 B：秩序守护型
女主站在“规则维护”立场，早期限制主角冒进，中期因局势恶化被迫破例，后期重建新秩序。  
主线影响：提升价值观冲突，主角成长不再单线爽推，读者会更关注人物立场碰撞。

### 方案 C：资源竞争型
女主与主角争夺同一核心机缘，合作仅是阶段策略，关键节点必须二选一。  
主线影响：节奏更快，爽点密度更高，适合连载期提高追更期待感。

## 风险与代价
- 方案 A 风险：若真相线推进过慢，会拖慢读者反馈节奏。
- 方案 B 风险：女主容易被写成“说教角色”，需增加行动证据。
- 方案 C 风险：处理不好会损伤 CP 预期，需要在冲突后给情感补偿场景。
