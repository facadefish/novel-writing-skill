# Commands Cheat Sheet

## Core Commands
- `novel scene simulate` / `novel ss`: multi-cast scene simulation with OOC guard.
- `novel outline view` / `novel ov`: outline board in vertical/horizontal/both views.

## Common Supporting Commands
- `novel discuss character` / `novel ch`
- `novel discuss world` / `novel wd`
- `novel discuss plot` / `novel pt`
- `novel apply decision` / `novel ok`
- `novel audit show` / `novel au`
- `novel rollback decision` / `novel rb`
- `novel asset check consistency` / `novel ck`

## Example
```bash
novel ss --event e018 --chars "linyan,hantu,zhifa" --scene "玄铁矿场夜审" --goal "夺钥符并立威" --ending "林砚带伤突围"
novel ov --book-id xh_demo_001 --scope chapter --target ch018 --view both
```
